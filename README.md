# 🚀 StartUp Forge - Client

StartUp Forge is a modern startup collaboration platform that connects founders and collaborators. Founders can create startup profiles, post opportunities, and build teams, while collaborators can discover startups and apply for roles that match their skills.

## 🌐 Live Demo

Add your deployed client URL here:
https://startupforge-client-swart.vercel.app
 

## ✨ Features

### 👨‍💼 Founder Features

- Create and manage startup profiles
- Post collaboration opportunities
- View and manage applicants
- Premium plan support for unlimited opportunity posting
- Founder dashboard with analytics

### 👨‍💻 Collaborator Features

- Browse startup opportunities
- Apply to opportunities
- Track application status
- Manage personal profile
- View startup details

### 🛡️ Authentication & Authorization

- Email & Password Authentication
- Google Sign-In
- JWT Protected Routes
- Role-Based Access Control
- Admin, Founder, and Collaborator Roles

### 📊 Admin Features

- Manage users
- Monitor platform activity
- Control user roles
- Review startups and opportunities

### 🎨 User Experience

- Fully Responsive Design
- Modern Dashboard UI
- Dark Mode Support
- Smooth Animations with Framer Motion
- Interactive Data Tables
- Pagination Support
- Loading & Error States
